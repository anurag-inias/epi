# Reverse all the words in a sentence

## Description

Given a sentence as a character array, reverse the order of the words.

## Stub

```java
public static void reverseWords(char[] input) {
  // Fill here.
}
```

## Solution

??? "Expand"

    First extract the reversal into a subroutine of its own.

    ```java
    private static void reverse(char[] input, int start, int end) {
      while (start < end) {
        char t = input[start];
        input[start++] = input[end];
        input[end--] = t;
      }
    }
    ```

    Next we call this twice over the input sentence:

    1. once over the whole sentence string.
    2. then once over each word.

    ```java
    public static void reverseWords(char[] input) {
      // reverse the whole string.
      reverse(input, 0, input.length - 1);

      // index to the start of a word.
      // -1 represents that we have not yet found a word to reverse.
      int s = -1;
      for (int e = 0; e < input.length; e++) {
        // we have reached the end of a word.
        if (input[e] == ' ') {
          // we had a word to reverse to begin with. 
          // this ensures that we don't try to reverse empty padding before / after
          // the sentence.
          if (s >= 0) {
            reverse(input, s, e - 1);
            s = -1;
          }
        } else {
          if (s == -1) s = e;
        }
      }
      // If the sentence has no starting / ending padding (most likely the case),
      // this makes sure we reverse the last word in such a sentence.
      if (s >= 0) reverse(input, s, input.length - 1);
    }
    ```
