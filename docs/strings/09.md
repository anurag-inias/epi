# Convert from Roman to decimal

## Description

Write a program to convert roman number in decimals

## Example

```
I =    1
V =    5
X =   10
L =   50
C =  100
D =  500
M = 1000
```

## Stub

```kotlin
fun romanToInteger(s: String): Int {
  // Fill here.
}
```

## Solution

??? "Expand"

    We need to check if the current digit is larger than previous digit or not.

    === "Current digit < previous digit"

        ```
        VI = 6
        ```

        `sum += curr`

    === "Current digit = previous digit"

        ```
        II = 2
        ```

        `sum += curr` again

    === "Current digit > previous digit"

        ```
        IV = 4
        ```

        `sum += curr - 2 * prev`

    ```kotlin
    fun romanToInteger(s: String): Int {
      var sum = 0
      var prev = 0
      for (c in s) {
        sum += c.romanToInt
        if (c.romanToInt > prev) {
          sum -= 2 * prev
        }
        prev = c.romanToInt
      }
      return sum
    }

    private val Char.romanToInt: Int
      get() = when(this) {
        'I' -> 1
        'V' -> 5
        'X' -> 10
        'L' -> 50
        'C' -> 100
        'D' -> 500
        'M' -> 1000
        else -> Int.MIN_VALUE
      }
    ```
