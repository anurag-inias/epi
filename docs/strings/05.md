# Test palindromicity

## Description

Implement a function which takes as input a string s and returns true if s is a palindromic string.

## Stub

```java
public static boolean isPalindromic(String s) {
  // Fill here.
}
```

## Solution

??? "Expand"

    ```java
    public static boolean isPalindromic(String s) {
      int start = 0;
      int end = s.length() - 1;
      while (start < end) {
        if (s.charAt(start++) != s.charAt(end--)) return false;
      }
      return true;
    }
    ```
