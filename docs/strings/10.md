# Compute all valid IP addresses

## Description

Given a string of made up of digits, list all valid IP addresses that can be created from it by adding `.`.

## Example

```
19216811 = 1.92.168.11
           19.2.168.11
           19.21.68.11
           19.216.8.11
           19.216.81.1
           192.1.68.11
           192.16.8.11
           192.16.81.1
           192.168.1.1
```

## Solution

??? "Expand"

    So let's first start with a helper method to figure out if a substring is a valid chunk of an IP address. That is, it lies inside $[0, 256)$ range. It also filters out certain invalid strings, e.g.

    1. `00`
    2. `012`

    ```kotlin
    private fun CharSequence.isChunkValid(): Boolean {
      var value = 0
      for (c in this) {
        value = 10 * value + c.digitToInt()
      }
      if (value == 0) return length == 1
      if (value !in 1 until 256) return false
      return this[0] != '0'
    }
    ```

    Next we utilize the typical backtracking cookbook for our helper function.

    ```
    fun helper(progressTracker, partialSolution) {
      ...
    }
    ```

    In our context, we have:

    1. `progressTracker` to be `index: Int` which is checking each char of the input string one-by-one.
    2. `partialSolution` to be two separate variables. `chunk: StringBuilder` which is one of the chunk of the whole IP address. and `partial: List<String>` which are chunks preceding current `chunk`.

    With this, our backtracking algorithms comes out to be:

    ```kotlin
    fun getValidIpAddress(s: String): List<String> {
      if (s.isEmpty()) return mutableListOf()
      val results = mutableListOf<String>()
      helper(s, 1, mutableListOf(), StringBuilder().append(s[0]), results)
      return results
    }

    private fun helper(
      s: String,
      index: Int,
      partial: MutableList<CharSequence>,
      chunk: StringBuilder,
      results: MutableList<String>
    ) {
      // [chunk 1, chunk 2, chunk 3] + current_chunk
      if (partial.size > 3) return

      // We have reached the end of the input.
      if (index == s.length) {
        // The last chunk is valid, and we have 3 preceding valid chunks.
        if (chunk.isChunkValid() && partial.size == 3) {
          results += partial.joinToString(separator = ".") + '.' + chunk
        }
        return
      }

      // option 1, start a new chunk, starting with the current character.
      partial += chunk
      helper(s, index + 1, partial, StringBuilder().append(s[index]), results)
      partial.removeLast()

      // option 2, expand chunk by appending current character.
      chunk.append(s[index])
      if (chunk.isChunkValid()) {
        helper(s, index + 1, partial, chunk, results)
      }
    }
    ```
