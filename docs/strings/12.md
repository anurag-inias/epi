# Implement run-length encoding

## Description

Implement run-length encoding and decoding functions. Assume the string to be
encoded consistsof letters of the alphabet, with no digits,and thestring to be decoded
is a valid encoding.

## Example

```
encode("aaaabcccaa") = "4alb3c2a"
decode("3e4f2e") = "eeeffffee"
```

        ```kotlin
        fun encoding(s: String): String {
          val encoded = StringBuilder()
          var char: Char? = null
          var count = 0

          for (c in s) {
            if (char == null) {
              char = c
              count = 1
            } else if (char == c) {
              count++
            } else {
              encoded.append(count).append(char)
              char = c
              count = 1
            }
          }

          return encoded.append(count).append(char).toString()
        }
        ```

        ```kotlin
          var decoded = StringBuilder()

          for (c in s) {
            if (c in '0'..'9') {
              count = 10 * count + c.digitToInt()
            } else {
        ```
