# Compute the spreadsheet column encoding

## Description

Convert a given excel sheet column name to its corresponding column number.

## Example

```
A   1
B   2
C   3
...
Z   26
AA  27
AB  28
```

## Stub

```kotlin
fun ssDecodeColID(col: String): Int {
  // Fill here.
}
```

## Solution

??? "Expand"

    ```kotlin
    fun ssDecodeColID(col: String): Int {
      var value = 0
      for (c in col) {
        value = 26 * value + (c - 'A' + 1)
      }
      return value
    }
    ```
