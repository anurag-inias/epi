# Replace and remove

## Description

Given an array of characters, perform the following operations:

1. Replace all `a` with `d, d`.
2. Remove all `b`.

You will also be given a number `size` which is the number of elements of the array on which this operation is to be applied. Ignore the remaining characters.

Assume that there is always enough space in the array to hold the post-processed content.

## Example

```
[a, b, a, c, _] => [d, d, d, d, c]
```

## Stub

```java
public static int replaceAndRemove(int size, char[] s) {
  // Fill here.
}
```

## Solution

??? "Explanation"

    First of all, you can very simply solve this problem with an auxiliary array. But that's not what we want. There is more to this problem than meets the eyes. It demonstrates two key ideas.

    First idea: we can **shift elements** to the left by omitting the processing of undesirable elements.

    ```java
    public static int replaceAndRemove(int size, char[] s) {
      // First pass.
      // Shift all elements to the left, overwriting 'b's.
      int cursor = 0; int aCount = 0;
      for (int i = 0; i < size; i++) {
        char c = s[i];
        if (c != 'b') s[cursor++] = c;
        if (c == 'a') aCount++;
      }
      // ... snippet
    }
    ```

    This loops ends with:

    ```
    [a, b, a, c, _] => [a, a, c, c, _]
                                 ^ cursor
    ```

    Now we will fill the array from right to left. Note that our cursor counted all the characters in the array except `b`, which are deleted. The space will need to hold all the elements with the `d`s would be `cursor + aCount`.

    ```java
    // ...snippet

    // Second pass.
    // Fill in 'd's.
    int from = cursor - 1;
    int to = from + aCount;
    while (from >= 0) {
      if (s[from] == 'a') {
        s[to--] = 'd';
        s[to--] = 'd';
      } else {
        s[to--] = s[from];
      }
      from--;
    }
    ```

??? "Full solution"

    ```java
    public static int replaceAndRemove(int size, char[] s) {
      // First pass.
      // Shift all elements to the left, overwriting 'b's.
      int cursor = 0; int aCount = 0;
      for (int i = 0; i < size; i++) {
        char c = s[i];
        if (c != 'b') s[cursor++] = c;
        if (c == 'a') aCount++;
      }

      // Second pass.
      // Fill in 'd's.
      int from = cursor - 1;
      int to = from + aCount;
      while (from >= 0) {
        if (s[from] == 'a') {
          s[to--] = 'd';
          s[to--] = 'd';
        } else {
          s[to--] = s[from];
        }
        from--;
      }

      // new size.
      return cursor + aCount;
    }
    ```
