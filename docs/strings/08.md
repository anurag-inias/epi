# The look-and-say problem

## Description

Write a program that takes as input an integer n and returns the nth integer in the
look-and-say sequence

## Example

```
1, 11, 21, 1211, 111221, 312211, 13112221, 1113213211
```

## Stub

```kotlin
fun lookAndSay(n: Int): String {
  // Fill here.
}
```

## Solution

??? "Expand"

    We just iterate over the string, counting the occurrences of a digit.

    ```kotlin
    fun lookAndSay(n: Int): String {
      var v: CharSequence = "1"
      for (i in 2..n) {
        v = verbalize(v)
      }
      return v.toString()
    }

    private fun verbalize(n: CharSequence): CharSequence {
      val verbalization = StringBuilder()

      var digit: Char? = null
      var count = 0
      for (c in n) {
        if (digit == null) {
          digit = c
          count = 1
        } else if (digit == c) {
          count++
        } else {
          verbalization.append(count).append(digit)
          digit = c
          count = 1
        }
      }
      verbalization.append(count).append(digit)

      return verbalization
    }
    ```
