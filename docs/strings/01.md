# Interconvert strings and integers

## Description

Write a program that converts `int` to and from `string`.

## Stub

```kotlin
fun intToString(x: Int): String {
  // Fill here.
}

fun stringToInt(s: String): Int {
  // Fill here.
}
```

## Solution

??? "Int to String"

    We need to use a helper method which takes in `Long` as opposed to `Int` so it can handle `-2147483648`.

    ```kotlin
    fun intToString(x: Int): String {
      if (x < 0) return "-" + helper(-x.toLong())
      return helper(x.toLong())
    }

    private fun helper(x: Long): String {
      var x = x
      val sb = StringBuilder()
      do { // (1)
        sb.append(x % 10)
        x /= 10
      } while (x != 0L)
      return sb.reversed().toString()
    }
    ```

    1. a `while` loop will simply convert `0` to `""`. So we use a `do-while`.


??? "String to Int"

    The book's solution does not account for `+` prefix.

    ```kotlin
    fun stringToInt(s: String): Int {
      if (s.isEmpty()) return 0 // (1)
      if (s[0] == '-') return -stringToInt(s.slice(1 until s.length)) // (2)
      if (s[0] == '+') return  stringToInt(s.slice(1 until s.length)) // (3)

      var value = 0L
      for (c in s) {
        value = 10 * value + c.digitToInt()
      }
      return value.toInt()
    }
    ```

    1. `"" -> 0`
    2. `"-123" -> -stringToInt("123")`
    3. `"+123" -> stringToInt("123")`
