Write a program which takes as input a phone number, specified as a string of digits,
and returns all possible character sequences that correspond to the phone number.
The cell phone keypad is specified by a mapping that takes a digit and returns the
corresponding set of characters. The character sequences do not have to be legal
words or phrases.

## Solution

??? "Expand"

    Backtracking-like solution, without backtracking.

    ```java
    public static List<String> phoneMnemonic(String phoneNumber) {
      List<String> mnemonic = new ArrayList<>();
      helper(phoneNumber, 0, new char[phoneNumber.length()], mnemonic);
      return mnemonic;
    }

    
    private static void helper(String phoneNumber, int index, char[] partial, List<String> results) {
      if (index == phoneNumber.length()) {
        results.add(new String(partial));
        return;
      }

      char[] mapping = MAPPING[phoneNumber.charAt(index) - '0'].toCharArray();
      for (char c : mapping) {
        partial[index] = c;
        helper(phoneNumber, index + 1, partial, results);
      }
    }

    // The mapping from digit to corresponding characters.
    private static final String[] MAPPING = {"0",   "1",   "ABC",  "DEF", "GHI",
        "JKL", "MNO", "PQRS", "TUV", "WXYZ"};
    ```
