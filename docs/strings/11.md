# Write a string sinusoidally

## Description

Write a program which takes as input a string $s$ and returns the sinusoidal representation of it.

## Example

```
s = "Hello_World!"
```

When we write it sinusoidally, we get:

```
 e   _   l
H l o W r d
   l   o   !
```

The three lines here become `e_l`, `HloWrd` and `lo!`. Appending all three, we get

```
e_lHloWrdlo!
```

## Stub

```kotlin
fun snakeString(s: String): String {
  // Fill here.
}
```

## Solution

??? "Expand"

    Let's make one this clear, we can use `(i+1) % 3` here for sinusoidal progression, since it's actuall spiral progression.

    Instead let's look at the index of the line getting the characters appended.

    ```
    1 0 1 2
            1 0 1 2
                    1 0 1 2
                            ....
    ```

    So we can just iterate over `1 0 1 2` over and over to get the target index for the line to get the next character.

    ```kotlin
    private val sequence = listOf(1, 0, 1, 2)

    fun snakeString(s: String): String {
      val lines = mutableListOf(StringBuilder(), StringBuilder(), StringBuilder())

      for ((i, c) in s.withIndex()) {
        lines[sequence[i % sequence.size]].append(c)
      }

      return lines.joinToString(separator = "")
    }
    ```
