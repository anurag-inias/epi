# Evaluate RPN expressions

## Description

Write a program that takes an arithmetical expression in RPN and returns the number
that the expression evaluates to. The token in the expression are separated by `,`.

## Stub

```kotlin
fun eval(expression: String): Int {
  // Fill here
}
```

## Solution

??? "Expand"

    ```java
    fun eval(expression: String): Int {
      val stack = ArrayDeque<Int>()
      for (token in expression.split(',')) {
        if (token.isOperator) {
          val b = stack.removeLast()
          val a = stack.removeLast()
          stack.add(token.operate(a, b))
        } else {
          stack.add(token.toInt())
        }
      }
      return stack.removeLast()
    }

    private fun String.operate(a: Int, b: Int): Int = when(this) {
      "+" -> a + b
      "-" -> a - b
      "*" -> a * b
      else -> a / b
    }

    private val String.isOperator: Boolean
      get() = length == 1 && this[0] !in '0'..'9'
    ```
