# Test a string over for well-formedness

## Description

Write a program that tests if a string made up of the characters `(`, `)`, `[`, is well-formed.

## Stub

```kotlin
fun isWellFormed(s: String): Boolean {
  // Fill here
}
```

## Solution

??? "Expand"

    ```kotlin
    fun isWellFormed(s: String): Boolean {
      val stack = ArrayDeque<Char>()
      for (c in s) {
        when {
          c.isOpening -> stack.add(c)
          stack.isNotEmpty() && stack.last() == c.opening -> stack.removeLast()
          else -> return false
        }
      }
      return stack.isEmpty()
    }

    private val Char.isOpening: Boolean
      get() = this == '(' || this == '[' || this == '{'

    private val Char.opening: Char
      get() = when(this) {
        ')' -> '('
        ']' -> '['
        else -> '{'
      }
    ```
