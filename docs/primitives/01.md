# Computing the parity of a word

## Description

Find parity of a given binary sequence.

## Example

```
 input = 0b1011
parity = 1

 input = 0b10001000
parity = 0
```

## Stub

```java
public static short parity(long x) {
  // return here
}
```

## Solution

??? "Expand"

    ```java
    public static short parity(long x) {
      int parity = 0;
      while (x != 0) {
        parity = 1 - parity;
        x = x & (x - 1);
      }
      return (short) parity;
    }
    ```
