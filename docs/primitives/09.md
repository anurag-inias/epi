# Reverse digits

## Description

Reverse the digits of an integer.

## Examples

```
 123 => 321
-314 => -413
```

## Stub

## Solution

??? "Expand"

    ```java
    public static long reverse(int x) {
      boolean negative = x < 0;
      x = x < 0 ? -x : x; // |x|

      long r = 0;         // tackle potential overflow
      while (x != 0) {
        r = r * 10 + x % 10;
        x /= 10;
      }
      return negative ? -r : r;
    }
    ```

## Unit tests

```kotlin
@Test
fun first() {
  assertThat(reverse(123)).isEqualTo(321)
  assertThat(reverse(-315)).isEqualTo(-513)

  assertThat(reverse(2147483647)).isEqualTo(7463847412) // Int.MAX_VALUE
}
```
