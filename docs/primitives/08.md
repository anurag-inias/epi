# Compute $x ^ y$ without arithmetical operators

## Description

Implement `pow` algorithm without using operator itself.

## Stub

```java
public static float pow(float x, int y) {
  // Fill me.
}
```

## Solution

??? "First approach"

    Repeated multiplication.

??? "Second approach"

    $$
    x^y = \begin{cases}
    (x^{\frac{y}{2}})^2 & \text{ if } y \text{ is even} \\
    x \times (x^{\frac{y}{2}})^2 & \text{ otherwise}
    \end{cases}
    $$

    ```kotlin
    fun pow(x: Float, y: Int): Float {
      if (y == 0) return 1F
      if (y < 0) {
        return pow(1/x, -y)
      }
      var result = 1F
      var a = x
      var p = y
      while (p != 0) {
        if (p % 2 != 0) {
          result *= a
        }
        a *= a
        p = p ushr 1
      }
      return result
    }
    ```

## Unit tests

```kotlin
@Test
fun first() {
  assertThat(pow(3F, 1)).isEqualTo(3F)
  assertThat(pow(3F, 2)).isEqualTo(9F)
  assertThat(pow(3F, 3)).isEqualTo(27F)
  assertThat(pow(3F, 4)).isEqualTo(81F)
}

@Test
fun second() {
  val epsilon = 0.00000001F
  assertThat(pow(1 / 3f, 1)).isEqualTo(1 / 3F)
  assertThat(abs(pow(1 / 3f, 2) - 1 / 9F)).isLessThan(epsilon)
  assertThat(abs(pow(1 / 3f, 3) - 1 / 27F)).isLessThan(epsilon)
  assertThat(abs(pow(1 / 3f, 4) - 1 / 81F)).isLessThan(epsilon)
}
```
