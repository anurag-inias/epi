# Check if a decimal integer is a palindrome

## Description

Check if an integer rads the same digits forwards and backwards.

## Examples

```
 123 => false
 121 => true
-121 => false
```

## Stub

```java
public static long isPalindromeNumber(int x) {
  // Fill me.
}
```

## Solution

??? "Expand"

    Use the [reverse digit](09.md) as subroutine.

    ```java
    public static boolean isPalindromeNumber(int x) {
      if (x < 0) return false;
      return x == reverse(x);
    }

    private static long reverse(int x) {
      boolean negative = x < 0;
      x = x < 0 ? -x : x;

      long r = 0;
      while (x != 0) {
        r = r * 10 + x % 10;
        x /= 10;
      }
      return negative ? -r : r;
    }
    ```
