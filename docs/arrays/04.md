# Advancing through an array

## Description

We are given an integer array $nums$ where $nums[i]$ represents how far you can jump from index $i$. Determine if we can reach the end of the array by jumping, starting from index $0$.

## Example

```
 Input = [3, 2, 0, 0, 2, 0, 1]
Output = true
```

## Stub

```java
public static boolean canReachEnd(List<Integer> maxAdvanceSteps) {
  // Fill here.
}
```

## Solution

??? "First approach"


    Maintain a boolean array to track which indices are reachable from index $0$. Iterate the list from left to right, and for each index $i$, look back at indices $[0, i-1]$ to see:

    1. if they are reachable from $0$, and
    2. if they allow jumping upto or beyond $i$.

    ```java
    public static boolean canReachEnd2(List<Integer> maxAdvanceSteps) {
      boolean[] reachable = new boolean[maxAdvanceSteps.size()];
      reachable[0] = true;

      for (int i = 1; i < maxAdvanceSteps.size(); i++) {
        // look at indices j in [0, i-1]
        for (int j = i-1; j >= 0; j--) {
          // if j itself is not reachable from 0, no point analyzing further. 
          if (!reachable[j]) continue;

          // Furthest you can get from j is = j + maxAdvanceSteps[j]
          // Does that include i?
          // If so, then i is reachable from 0 too.
          int dist = i - j;
          if (maxAdvanceSteps.get(j) >= dist) {
            reachable[i] = true;
            break;
          }
        }
      }
      return reachable[maxAdvanceSteps.size() - 1];
    }
    ```

??? "Second approach"

    Actually, we don't need the extra space. We can simply process the list left to right, keeping track of furthest index reachable, say $f$.

    ```java
    public static boolean canReachEnd(List<Integer> maxAdvanceSteps) {
      int n = maxAdvanceSteps.size();
      int f = 0;

      for (int i = 0; i < n - 1 && i <= f; i++) { // (1)
        f = Math.max(f, i + maxAdvanceSteps.get(i));
        if (f >= n-1) return true;
      }
      return false;
    }
    ```

    1. notice the additional condition check $i \le f$. We only consider index $i$ if it's within the farthest position we can already reach.
