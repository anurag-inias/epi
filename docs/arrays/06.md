# Buy and sell a stock once

## Description

Given a list containing daily prices of a given stock. Calculate the maximum profit we can make if buying-selling stock once.

## Example

```
 Input = [310, 315, 275, 295, 260, 270, 290, 230, 255, 250]
Output = 290 - 260 = 30
```

## Stub

```java
public static double computeMaxProfit(List<Double> prices) {
  // Fill here.
}
```

## Solution

??? "Expand"

    Simply iterate left to right while maintaining the maximum profit seen and minimum price seen.

    At each step calculate the maximum possible profit by diffing the current price and the minimum price seen.

    ```java
    public static double computeMaxProfit(List<Double> prices) {
      double profit = 0, min = prices.getFirst();

      for (int i = 1; i < prices.size(); i++) {
        profit = Math.max(profit, prices.get(i) - min);
        min = Math.min(min, prices.get(i));
      }

      return profit;
    }
    ```
