# Increment an arbitrary-precision integer

## Description

Given a list, represending the digits of a arbitrary range integer, increment it by one.

## Example

```
 Input = [1, 2, 9]
Output = [1, 3, 0]
```

    ```java
    public static List<Integer> plusOne(List<Integer> A) {
      int carry = 1;
      int digit = A.size() - 1;

      while (carry != 0) {
        if (digit < 0) {
          A.add(0, carry);
          break;
        }
        int sum = A.get(digit) + carry;
        A.set(digit, sum % 10);
        carry = sum / 10;
        digit--;
      }
      return A;
    }
    ```
