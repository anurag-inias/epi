# Test for overlapping lists - lists are cycle-free

## Description

Write a program that takes two cycle-free singly linked lists, and determines if there
exists a node that is common to both lists.

## Stub

```java
public static ListNode<Integer> overlappingNoCycleLists(ListNode<Integer> l0, ListNode<Integer> l1) {
  // Fill here
}
```

## Solution

??? "Expand"

    Let's assume the first list is of length `a+c` and second list of length `b+c`, where `c` is the length of the overlap. We can find $|(a+c) - (b+c)|$ the steps we need to take in either lists before we are on track to have both cursor the same distance from the joint.

    ```java
    public static ListNode<Integer>
    overlappingNoCycleLists(ListNode<Integer> l0, ListNode<Integer> l1) {
      var ac = length(l0);
      var bc = length(l1);
      while (ac > bc) {
        l0 = l0.next;
        ac--;
      }
      while (bc > ac) {
        l1 = l1.next;
        bc--;
      }
      while (l0 != l1) {
        l0 = l0.next;
        l1 = l1.next;
      }
      return l0;
    }

    private static <T> int length(ListNode<T> head) {
      int len = 0;
      while (head != null) {
        len++;
        head = head.next;
      }
      return len;
    }
    ```
