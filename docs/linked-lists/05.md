# Test for overlapping lists - lists may have cycles

## Description

Same as before, but now both of the lists can have cycle.

## Stub

```java
public static ListNode<Integer> overlappingLists(ListNode<Integer> l0,
      ListNode<Integer> l1) {
  // Fill here
}
```

## Solution

??? "Expand"

    We have to consider 4 key cases here:

    1. Neither lists have a cycle: in this case the solution reduces to the previous problem.
    2. One list has a cycle but the other doesn't: in which case they can't have any shared nodes.
    3. Both lists have shared cycles: in which we need to find the node connecting either of the list to the cycle.
    4. Both lists have cycles but disjoint: in which case there are no shared nodes again.

    === "Solution"

        ```java
        public static ListNode<Integer> overlappingLists(ListNode<Integer> l0,
            ListNode<Integer> l1) {
          var n0 = cycleNode(l0);
          var n1 = cycleNode(l1);
          if (n0 == null && n1 == null) {
            // neither has a cycle.
            return overlappingNoCycleLists(l0, l1);
          }
          if (n0 != null && n1 != null) {
            // both have a cycle.

            // We make sure that these are not two different cycles, completely
            // disjoint from each other.
            var cursor = n0;
            boolean foundInFirstList = false;
            do {
              if (cursor == n1) foundInFirstList = true;
              cursor = cursor.next;
            } while (cursor != n0 && !foundInFirstList);
            // Node found in second cycle is absent from first cycle, 
            // meaning these lists are disjoint.
            if (!foundInFirstList) return null;

            // Finding the node which connects the first list to its cycle.
            var front = l0;
            int l = cycleLength(n0);
            while (l-- > 0) front = front.next;
            var back = l0;
            while (front != back) {
              front = front.next;
              back = back.next;
            }
            return front;
          }
          // one has a cycle but not the other.
          return null;
        }
        ```

    === "Utlities"

        ```java
        // Find the length of a linked list assuming it has no cycle.
        private static <T> int length(ListNode<T> head) {
          int len = 0;
          while (head != null) {
            len++;
            head = head.next;
          }
          return len;
        }

        // For a linked list with cycle, find the length of said cycle.
        private static <T> int cycleLength(ListNode<T> node) {
          var cursor = node;
          int length = 0;
          do {
            length++;
            cursor = cursor.next;
          } while (cursor != node);
          return length;
        }

        // Common boundary check used while checking for cyclicity.
        private static <T> boolean hasHorizon(ListNode<T> head) {
          return head != null && head.next != null;
        }

        // Checks if a list has a cycle or not. Returns null node if no cycle.
        // Returns one of the node within the cycle if there is one.
        private static <T> ListNode<T> cycleNode(ListNode<T> head) {
          if (!hasHorizon(head)) return null;
          var slow = head;
          var fast = head;
          do {
            slow = slow.next;
            fast = fast.next.next;
          } while (hasHorizon(fast) && slow != fast);
          return slow == fast ? slow : null;
        }
        ```

    === "Helper - Find common node when no cycles"

        ```java
        public static <T> ListNode<T>
        overlappingNoCycleLists(ListNode<T> l0, ListNode<T> l1) {
          var ac = length(l0);
          var bc = length(l1);
          while (ac > bc) {
            l0 = l0.next;
            ac--;
          }
          while (bc > ac) {
            l1 = l1.next;
            bc--;
          }
          while (l0 != l1) {
            l0 = l0.next;
            l1 = l1.next;
          }
          return l0;
        }
        ```
