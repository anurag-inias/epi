# Implement list pivoting

## Description

For any integer $k$, the pivot of a list of integers with respect to $k$ is that list with its nodes reordered so that all nodes containing keys less than $k$ appear before nodes containing $k$, and all nodes containing keys greater than $k$ appear after the nodes
containing $k$.

    public static ListNode<Integer> listPivoting(ListNode<Integer> l, int x) {
      ListNode<Integer> lh = null, lt = null, eh = null, et = null, gh = null, gt = null;
      while (l != null) {
        var next = l.next;
        l.next = null;

        switch (Integer.compare(l.data, x)) {
          case -1 -> {
            if (lh == null) lh = lt = l;
            else lt = lt.next = l;
          }
          case 0 -> {
            if (eh == null) eh = et = l;
            else et = et.next = l;
          }
          case 1 -> {
            if (gh == null) gh = gt = l;
            else gt = gt.next = l;
          }
        }
        l = next;
      }

      if (lt != null) lt.next = eh != null ? eh : gh; // if the middle list is empty, join with the last list instead.
      if (et != null) et.next = gh;

      return lh != null ? lh : (eh != null ? eh : gh);
    }
