# Test whether a singly linked list is palindromic

## Description

Write a program that tests whether a singly linked list is palindromic.

## Stub

```java
public static boolean isLinkedListAPalindrome(ListNode<Integer> L) {
  // Fill here
}
```

## Solution

??? "Expand"

    We run two pointers, one slow (one step at a time) and another fast (two step at a time) to find the middle of the list.

    Once we do that, we split the list in two part and compare the two chunks (after reversing the second half).

    ```java
    public static boolean isLinkedListAPalindrome(ListNode<Integer> L) {
      if (L == null || L.next == null) return true;

      ListNode<Integer> prev = null;
      var halfway = L;
      var scout = L;
      while (scout != null && scout.next != null) {
        prev = halfway;
        halfway = halfway.next;
        scout = scout.next.next;
      }
      boolean lengthEven = scout == null;

      var firstHalf = L;
      prev.next = null;
      var secondHalf = lengthEven ? halfway : halfway.next;

      return areSame(firstHalf, reverse(secondHalf));
    }

    private static <T> ListNode<T> reverse(ListNode<T> head) {
      ListNode<T> prev = null;
      while (head != null) {
        var next = head.next;
        head.next = prev;
        prev = head;
        head = next;
      }
      return prev;
    }

    private static <T> boolean areSame(ListNode<T> first, ListNode<T> second) {
      while (first != null && second != null) {
        if (!first.data.equals(second.data)) return false;
        first = first.next;
        second = second.next;
      }
      return first == second; // both must be null
    }
    ```
