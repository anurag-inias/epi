# Merge two sorted lists

## Description

Write a program that merges given sorted linked lists.

## Stub

```java
public static ListNode<Integer> mergeTwoSortedLists(
  ListNode<Integer> L1,
  ListNode<Integer> L2) {
  // Fill here
}
```

    Fairly straighforward.

    ```java
    public static ListNode<Integer> mergeTwoSortedLists(ListNode<Integer> L1,
                                                        ListNode<Integer> L2) {
      if (L1 == null) return L2;
      if (L2 == null) return L1;
      ListNode<Integer> head = null, tail = null;
      while (L1 != null || L2 != null) {
        int u = L1 == null ? Integer.MAX_VALUE : L1.data;
        int v = L2 == null ? Integer.MAX_VALUE : L2.data;
        if (u < v) {
          if (head == null) head = tail = L1;
          else tail = tail.next = L1;
          L1 = L1.next;
        } else {
          if (head == null) head = tail = L2;
          else tail = tail.next = L2;
          L2 = L2.next;
        }
      }
      return head;
    }
    ```
