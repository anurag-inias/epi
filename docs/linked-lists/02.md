# Reverse a single sublist

## Description

Write a program that reverses the specified range of a linked list. The range is passed as two integer `start` and `finish`, inclusive, with numbering beginning at $1$.

## Stub

```kotlin
fun reverseSublist(
    L: ListNode<Int>?, start: Int,
    finish: Int
  ): ListNode<Int>? {
  // Fill here
}
```

## Solution

??? "Expand"

    Let's first sanitize our parameters, since we want the indices starting at `0` and not `1`. Also, we will be operating in Kotlin so we will move all our code to a kotlin method which expects non-null inputs.

    ```kotlin
    fun reverseSublist(
      L: ListNode<Int>?, start: Int,
      finish: Int
    ): ListNode<Int>? {
      if (L == null) return L
      return helper(L, start - 1, finish - 1)
    }
    ```

    Next, we need a helper method to reverse a linked list, starting from the given node.

    ```kotlin
    private fun <T> ListNode<T>.reverse(previous: ListNode<T>? = null, length: Int): ListNode<T> {
      if (length == 0) return this // (1)

      var l = length
      val futureTail = this // (2)
      var prev = previous
      var curr: ListNode<T>? = this

      while (curr != null && l-- > 0) {
        val next = curr.next
        curr.next = prev
        prev = curr
        curr = next
      }

      previous?.next = prev
      futureTail.next = curr
      return prev!!
    }
    ```

    1. nothing to do when section to reverse has length $0$.
    2. once this section is reversed, this will become the new tail of the sublist.

    Now all we need to do is call this on the node at index `start`.

    ```kotlin
    fun helper(
      L: ListNode<Int>, start: Int,
      finish: Int
    ): ListNode<Int> {
      var start = start
      val length = finish - start + 1

      // Head is being affected.
      if (start == 0) {
        return L.reverse(previous = null, length)
      }

      var prev: ListNode<Int>? = null
      var curr: ListNode<Int>? = L
      while (curr != null && start-- > 0) {
        prev = curr
        curr = curr.next
      }
      val unused = curr?.reverse(prev, length)
      return L
    }
    ```
