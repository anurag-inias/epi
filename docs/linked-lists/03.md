# Test for cyclicity

## Description

Write a program that takes the head of a singly linked list and returns null if there
does not exist a cycle, and the node at the start of the cycle, if a cycle is present.

## Stub

```java
public static ListNode<Integer> hasCycle(ListNode<Integer> head) {
  // Fill here
}
```

## Solution

??? "Expand"

    We start with a helper method for our boundary check.

    ```java
    private static <T> boolean hasHorizon(ListNode<T> node) {
      return node != null && node.next != null;
    }
    ```

    Rest is pretty straightforward.

    ```java
    public static ListNode<Integer> hasCycle(ListNode<Integer> head) {
      if (!hasHorizon(head)) return null;

      // First check if it has a cycle or not.
      var slow = head;
      var fast = head;
      do {
        slow = slow.next;
        fast = fast.next.next;
      } while (hasHorizon(fast) && slow != fast);
      if (!hasHorizon(fast)) return null;

      // Now find the length of the cycle.
      var length = 0;
      do {
        slow = slow.next;
        length++;
      } while (slow != fast);

      // Prep a rope of size [length].
      fast = head;
      while (length-- > 0) {
        fast = fast.next;
      }

      // Stuff it in the hoze.
      slow = head;
      while (slow != fast) {
        slow = slow.next;
        fast = fast.next;
      }

      return slow;
    }
    ```
