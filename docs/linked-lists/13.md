# Add list-based integers

## Description

Write a program which takes two singly linked lists of digits, and returns the list
corresponding to the sum of the integers they represent. The least significant digit
comes first.

## Solution

    ```java
    public static ListNode<Integer> addTwoNumbers(ListNode<Integer> L1,
                                                  ListNode<Integer> L2) {
      if (L1 == null) return L2;
      if (L2 == null) return L1;
      ListNode<Integer> head = null, tail = null;

      int carry = 0;
      while (L1 != null || L2 != null || carry != 0) {
        int sum = (L1 == null ? 0 : L1.data) + (L2 == null ? 0 : L2.data) + carry;

        if (head == null) head = tail = new ListNode<>(sum % 10, null);
        else tail = tail.next = new ListNode<>(sum % 10, null);

        carry = sum / 10;
        L1 = (L1 == null) ? null : L1.next;
        L2 = (L2 == null) ? null : L2.next;
      }

      return head;
    }
    ```
