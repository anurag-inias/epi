# Implement cyclic right shift for singly linked lists

## Description

Write a program that takes as input a singly linked list and a non-negative integer $k$,
and returns the list cyclically shifted to the right by $k$.

## Stub

```java
public static ListNode<Integer> cyclicallyRightShiftList(ListNode<Integer> L,
                                                           int k) {

}
```

## Solution

    ```java
    public static ListNode<Integer> cyclicallyRightShiftList(ListNode<Integer> L,
                                                            int k) {
      if (L == null || L.next == null) return L; // (1)

      // Find out the tail of the existing list.
      // This is needed because we'll need to stitch back 
      // elements here.
      var l = 1;
      var tail = L;
      while (tail.next != null) {
        l++;
        tail = tail.next;
      }
      // Account for k being large or in opposite direction.
      k = (l + (l - k) % l) % l;
      if (k == 0) return L;

      var head = L;
      ListNode<Integer> prev = null;
      while (k-- > 0) {
        prev = head;
        head = head.next;
      }
      prev.next = null;
      tail.next = L;
      return head;
    }
    ```

    1. if list is empty or just have one element, all rotation will be no-op.
