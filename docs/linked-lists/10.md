# Implement even-odd merge

## Description

Consider a singly linked list whose nodesare numbered starting at $0$. Define the even-
odd merge of the list to be the list consisting of the even-numbered nodes followed
by the odd-numbered nodes.

    ```java
    public static ListNode<Integer> evenOddMerge(ListNode<Integer> L) {
      if (L == null || L.next == null) return L;

      // cursor for even head, even tail, odd head and odd tail.
      ListNode<Integer> eh = null, et = null, oh = null, ot = null;
      // whether to put the next element in even list.
      boolean inEven = true;
      while (L != null) {
        var next = L.next; // (1)
        L.next = null;

        if (inEven) {
          if (eh == null) eh = et = L;
          else et = et.next = L;
        } else {
          if (oh == null) oh = ot = L;
          else ot = ot.next = L;
        }

        inEven = !inEven;
        L = next;
      }
      et.next = oh;
      return eh;
    }
    ```
