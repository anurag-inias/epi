# Remove the kth last element from a list

## Description

Given a singly linked list and an integer $k$, write a program to remove the $k^{th}$ last
element from the list. Your algorithm cannot use more than a few words of storage,
regardless of the length of the list. In particular, you cannot assume that it is possible
to record the length of the list.

    ```java
    // Assumes L has at least k nodes, deletes the k-th last node in L.
    public static ListNode<Integer> removeKthLast(ListNode<Integer> L, int k) {
      var front = L;
      while (front != null && k-- > 0) {
        front = front.next;
      }
      if (front == null) return L.next;

      ListNode<Integer> prev = null;
      var back = L;
      while (front != null) {
        prev = back;
        front = front.next;
        back = back.next;
      }
      prev.next = back.next;
      return L;
    }
    ```
